# 5.04 - Target Config (Exercise Target Configuration)

> **Navigation:**
> - FROM: Exercise Picker (2.01, Adding Mode) ‚Üí Smart Toast ‚Üí [Configure] ‚Üí **Target Config**
> - FROM: Exercise Picker (2.01, Replace Mode) ‚Üí Tap Exercise ‚Üí **Target Config**
> - FROM: Plan Builder Days (5.03) ‚Üí Tap Exercise Row ‚Üí **Target Config**
> - TO: Exercise Picker (2.01) (via "Save Targets" when opened from Picker)
> - TO: Plan Builder Days (5.03) (via "Save Targets" when opened from Plan Builder)
>
> **Note:** Target Config is now **optional** during exercise adding. Exercises are added
> immediately with smart defaults. Users can configure later via toast or by tapping
> the exercise row in Plan Builder Days.
>

## 2. Human-Readable Summary

**User Behavior**

The user configures how many sets, reps, and weight for an exercise they're adding or editing in their workout plan.

**Two Modes:**
- **Simple View (Default)**: For uniform targets (same reps/weight across all sets). Covers 80% of users (Newbies + most Gym Rats).
- **Advanced View**: For per-set customization (pyramid sets, drop sets). Unlocked via "Customize per set" link.

**Key Cases**
*   **Adding New Exercise**: Opens with smart defaults (history ‚Üí AI suggestion ‚Üí fallback: 3 sets √ó 10 reps @ Bodyweight).
*   **Editing Existing Exercise (Uniform)**: Opens in Simple View with current values.
*   **Editing Existing Exercise (Varied)**: Auto-opens Advanced View (per-set data can't be represented in Simple View).
*   **Bodyweight Toggle**: For exercises without external weight (push-ups, pull-ups, dips).
*   **Progressive Disclosure**: Newbies see Simple View only. Gym Rats can unlock Advanced with one tap.

**Data Displayed**
*   **Exercise Name & Category**: Context ("Bench Press ‚Ä¢ Chest").
*   **Sets**: Slider (1-10 range).
*   **Reps**: Slider (1-50 range).
*   **Weight**: Number input (opens **GymNumpad** component).
*   **Bodyweight Toggle**: Switches between weighted / bodyweight modes.

**Interaction & Motion**
Calm, premium feel. Sliders use spring physics (satisfying tactile feedback). Weight input opens **Custom Numpad** with slot-machine value animation and glowing accents. Transition from Simple ‚Üí Advanced uses liquid morph expansion (400ms spring). Save button ripples on success. All interactions have haptic feedback (light for inputs, medium for primary actions).
