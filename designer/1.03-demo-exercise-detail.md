## 2. Human-Readable Summary
**User Behavior**
The user lands on the exercise detail screen to log their first set. This is a "demo" version of the real screen they will use later. They see a helpful tip banner explaining what to do. Once they log a single set, the app automatically celebrates and moves them to the next step.

**Key Cases**
*   **First Time Experience**: The screen always treats this as the first time doing the exercise, showing a "First time!" message and empty input fields.
*   **Auto-Navigation**: Logging one set triggers an automatic transition to the success screen.

**Data Displayed**
*   **Exercise Name**: (e.g., "Bench Press")
*   **Tip Banner**: "This is where you'll log all your sets in the gym. Try logging one!"
*   **Input Fields**: Weight and Reps (empty).
